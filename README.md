# sparrow_utils

常用的工具类包。

* SPCollectionUtils
* SPDateUtils，使用时需要[初始化本地化语言](https://flutter.dev/docs/development/accessibility-and-localization/internationalization)
* SPEnumUtils
* SPStringUtils


## 使用

添加依赖：

```yaml
dependencies:
  flutter:
    sdk: flutter
  sparrow_utils: ^0.4.0
```

使用：

```dart
import 'package:sparrow_utils/sparrow_utils.dart

void main() {

  // SPEnumUtils...
  // SPStringUtils....

  // SPCollectionUtils....
  // SPDateUtils....
}

````
